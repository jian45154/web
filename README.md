Personal portfolio for **金苇航 Ian**, designed and built by **Lucien Auregin**.

## Live site

[jian45154.github.io](https://jian45154.github.io/)

## Local preview

Serve the folder with a static file server so the browser can load the JSON content:

```powershell
python -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Structure

- `index.html` - single-page scrolling portfolio
- `details.html` - legacy redirect to the work section
- `styles.css` - responsive visual design
- `script.js` - bilingual content switching
